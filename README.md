# Jogo De Adivinhacao


![](https://imgur.com/NVedEEa.gif)

## Introdução
Jogo de adivinhação que permite selecionar o nivel de dificuldade e um determinado numero de tentativas de acertar o numero gerado aleatoriamente.

## Funcionalidades
- Selecionar nivel de dificuldade.
- Digitar tentativas conforme nivel de dificuldade.
- Não permite usar numeros repetidos ao chutar.
- Não desconta tentativas ao ser bloqueado por usar número repetido.
- Sistema de pontuação onde o usuario começará com 1000 pontos e caso erre, será subtraido um valor da pontuação.
